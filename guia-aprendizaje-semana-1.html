Errores Comunes y Soluciones

❌ Error: "python no se reconoce como comando"

Causa: Python no está en el PATH del sistema

Solución:

  1. Desinstalar Python
  2. Volver a instalar marcando la opción "Add Python to PATH" en la primera pantalla
  3. O agregar manualmente al PATH:
    • Windows: Buscar "Variables de entorno" → PATH → Agregar ruta de Python (ej: C:\Python312)

❌ Error: "pip no se reconoce como comando"

Solución:

  • Intenta con: python -m pip install flask
  • O reinstala Python asegurándote de incluir pip

❌ Error: "ModuleNotFoundError: No module named 'flask'"

Causa: Flask no está instalado o está en un entorno virtual diferente

Solución:

  • Ejecutar: pip install flask
  • Verificar con: pip list (Flask debe aparecer en la lista)

❌ Error: "IndentationError: unexpected indent"

Causa: Python es muy estricto con la indentación (espacios al inicio de línea)

Solución:

  • Usar siempre 4 espacios para indentar (no Tab)
  • En VSCode, configurar: File → Preferences → Settings → buscar "Tab Size" → poner 4
  • Revisar que todo el código tenga la misma indentación

❌ Error: "SyntaxError: invalid syntax"

Causas comunes:

  • Olvidaste los dos puntos : después de if, for, while, def
  • Falta cerrar paréntesis, corchetes o comillas
  • Estás usando Python 2 en vez de Python 3 (print sin paréntesis)

❌ Error: "Address already in use" (Flask)

Causa: Ya hay una aplicación Flask corriendo en el puerto 5000

Solución:

  • Cerrar el terminal anterior o detener Flask con Ctrl+C
  • O usar otro puerto: app.run(debug=True, port=5001)

Ejemplo 3: Ciclos - Registro de Múltiples Estudiantes

Sistema que permite registrar varios estudiantes y calcular estadísticas del grupo. 

# ejemplo3_ciclos.py
# Sistema de registro múltiple con estadísticas

print("=" * 70)
print("   SISTEMA DE REGISTRO DE ESTUDIANTES - CON ESTADÍSTICAS")
print("=" * 70)

# Listas para almacenar información
nombres = []
promedios = []
grados = []

# Contador de estudiantes
contador = 0

# Ciclo para registrar estudiantes
while True:
    print(f"\n--- REGISTRO #{contador + 1} ---")
    nombre = input("Nombre del estudiante: ")
    grado = input("Grado (ej: 5to Primaria): ")
    promedio = float(input("Promedio: "))
    
    # Agregar a las listas
    nombres.append(nombre)
    grados.append(grado)
    promedios.append(promedio)
    contador += 1
    
    # Preguntar si desea continuar
    continuar = input("\n¿Registrar otro estudiante? (s/n): ").lower()
    if continuar != 's':
        break

# Calcular estadísticas
if contador > 0:
    promedio_grupal = sum(promedios) / len(promedios)
    promedio_mas_alto = max(promedios)
    promedio_mas_bajo = min(promedios)
    
    # Encontrar mejor y peor estudiante
    indice_mejor = promedios.index(promedio_mas_alto)
    indice_menor = promedios.index(promedio_mas_bajo)
    
    mejor_estudiante = nombres[indice_mejor]
    estudiante_bajo = nombres[indice_menor]
    
    # Contar aprobados y reprobados
    aprobados = sum(1 for p in promedios if p >= 60)
    reprobados = contador - aprobados
    
    # Contar por categoría
    sobresalientes = sum(1 for p in promedios if p >= 90)
    muy_buenos = sum(1 for p in promedios if 80 <= p < 90)
    buenos = sum(1 for p in promedios if 70 <= p < 80)
    regulares = sum(1 for p in promedios if 60 <= p < 70)
    insuficientes = sum(1 for p in promedios if p < 60)
    
    # Mostrar resultados
    print("\n" + "=" * 70)
    print("           LISTADO COMPLETO DE ESTUDIANTES")
    print("=" * 70)
    
    for i in range(contador):
        estado = "✓ APROBADO" if promedios[i] >= 60 else "✗ REPROBADO"
        print(f"{i+1}. {nombres[i]:<30} {grados[i]:<15} {promedios[i]:>6.2f} {estado}")
    
    print("\n" + "=" * 70)
    print("           ESTADÍSTICAS DEL GRUPO")
    print("=" * 70)
    print(f"Total de estudiantes registrados: {contador}")
    print(f"Promedio general del grupo: {promedio_grupal:.2f}")
    print(f"\n📊 Distribución por categoría:")
    print(f"  🌟 Sobresaliente (90-100): {sobresalientes}")
    print(f"  😊 Muy Bueno (80-89): {muy_buenos}")
    print(f"  👍 Bueno (70-79): {buenos}")
    print(f"  ⚠️  Regular (60-69): {regulares}")
    print(f"  ❌ Insuficiente (<60): {insuficientes}")
    print(f"\n✓ Aprobados: {aprobados}")
    print(f"✗ Reprobados: {reprobados}")
    print(f"\n🏆 Mejor estudiante: {mejor_estudiante} ({promedio_mas_alto:.2f})")
    print(f"⚠️  Requiere apoyo: {estudiante_bajo} ({promedio_mas_bajo:.2f})")
    print("=" * 70)
else:
    print("\nNo se registraron estudiantes.")

PARTE 2: PREPARACIÓN PARA EL PROYECTO INTEGRADOR

Sistema de Registro y Seguimiento de Estudiantes en Riesgo Educativo

A lo largo del curso desarrollarás una aplicación web con Flask que permita a docentes registrar y dar seguimiento a estudiantes que presentan indicadores de riesgo educativo (bajo rendimiento, ausentismo, problemas socioeconómicos, etc.). El sistema categorizará automáticamente a los estudiantes según el nivel de riesgo y facilitará la toma de decisiones para intervenciones oportunas.

Este proyecto tiene relevancia social directa en el contexto guatemalteco, donde muchas escuelas rurales carecen de sistemas digitales para monitorear el progreso estudiantil.
Antes de comenzar a programar, piensa en las siguientes preguntas (escribe tus respuestas en una libreta):

Análisis del problema:
- ¿Qué información es esencial registrar de cada estudiante?
- ¿Qué indicadores pueden señalar que un estudiante está en riesgo?
- ¿Cómo podría un docente usar este sistema en su día a día?

Diseño de la solución:
- ¿Qué páginas o secciones debería tener la aplicación?
- ¿Cómo organizarías la información visualmente?
- ¿Qué acciones principales debe poder hacer el docente?
