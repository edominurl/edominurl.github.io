Descripción del Curso

Este curso intensivo de 4 semanas está diseñado para formar futuros docentes de tecnología en los principios fundamentales del pensamiento computacional y la programación funcional con Python. A diferencia de un enfoque tradicional de ciencias de la computación, este programa prioriza la aplicación práctica y el impacto social, preparando a los estudiantes para enseñar programación de manera contextualizada y significativa.

Los estudiantes aprenderán a desarrollar aplicaciones web funcionales utilizando Flask, con un enfoque en la resolución de problemas educativos reales de Guatemala. El curso combina competencias técnicas esenciales con estrategias pedagógicas, permitiendo a los futuros docentes no solo programar, sino también diseñar experiencias de aprendizaje efectivas para sus estudiantes.

Modalidad Intensiva

  • Sesiones sincrónicas: Sábados 5h 45min (práctica supervisada + retroalimentación inmediata)
  • Trabajo autónomo: 8-10 horas semanales con guías de aprendizaje
  • Enfoque funcional: Conceptos aplicables inmediatamente sobre abstracciones complejas

Metodología

  • Aprendizaje invertido adaptado
  • Aprendizaje basado en proyectos con impacto social
  • Práctica reflexiva docente
  • Aprendizaje entre pares

Competencias e Indicadores de Logro

COMPETENCIA GENERAL

Desarrollar aplicaciones web funcionales utilizando Python y Flask para resolver problemas educativos y sociales del contexto guatemalteco, integrando pensamiento computacional con estrategias pedagógicas efectivas para la enseñanza de la programación.

Competencias Específicas:

1. Técnica-Funcional

Aplicar fundamentos de Python (variables, estructuras de control, funciones, manejo de datos) y Flask (rutas, templates, formularios) para crear aplicaciones web que resuelvan problemas reales, con manejo básico de datos y validaciones.

2. Pedagógica-Didáctica

Diseñar estrategias didácticas contextualizadas para enseñar programación a estudiantes de educación media, reflexionando sobre métodos efectivos de enseñanza del pensamiento computacional y adaptando contenidos técnicos a diferentes niveles.

3. Social-Reflexiva

Identificar problemas sociales y educativos del contexto guatemalteco que pueden abordarse mediante soluciones tecnológicas, considerando las implicaciones éticas del manejo de datos y la inclusión digital.

Indicadores de Logro:

  • Implementa algoritmos utilizando estructuras de control y funciones modulares en Python
  • Desarrolla aplicaciones web con Flask integrando rutas dinámicas y templates
  • Gestiona datos mediante archivos JSON y SQLite (CRUD completo)
  • Valida datos de entrada y maneja errores apropiadamente
  • Crea interfaces web funcionales con HTML, CSS y Bootstrap
  • Documenta procesos técnicos con enfoque pedagógico
  • Reflexiona críticamente sobre implicaciones éticas de soluciones tecnológicas

Proyecto Integrador

"Sistema de Seguimiento de Estudiantes en Riesgo Académico"

Los estudiantes desarrollarán una aplicación web completa para identificar y dar seguimiento a estudiantes en situación de riesgo de deserción escolar o bajo rendimiento académico. Este sistema permitirá a docentes y directores tomar decisiones informadas y realizar intervenciones oportunas.

JUSTIFICACIÓN SOCIAL

La deserción escolar es un problema crítico en Guatemala, especialmente en áreas rurales. Este sistema permite identificar tempranamente señales de alerta (inasistencias frecuentes, bajo rendimiento, situaciones familiares difíciles) y documentar las intervenciones realizadas. Es una herramienta práctica que los estudiantes podrían implementar en sus futuras escuelas.

FASES DEL PROYECTO:

1
Semana 1

Estructura base y lógica: Aplicación Flask con rutas, templates y funciones Python usando diccionarios y listas. Interfaz simple para mostrar información.

2
Semana 2

Persistencia de datos: Almacenamiento en archivos JSON y SQLite con operaciones básicas (INSERT y SELECT). Sistema de categorización por nivel de riesgo (verde, amarillo, rojo).

3
Semana 3

Interactividad y visualización: Formularios HTML funcionales con validación. Visualización mediante colores y gráficas simples. CRUD completo implementado.

4
Semana 4

Refinamiento y preparación: Manejo de errores, documentación completa, plan de clase para enseñanza, empaquetado para despliegue local o PythonAnywhere.

PRODUCTO FINAL:

  • Base de datos SQLite operativa
  • Interfaz web responsive con alertas visuales
  • Dashboard con estadísticas y visualizaciones

Mensaje Final del Docente

Estimados futuros docentes de tecnología:

Este curso intensivo será desafiante. En solo 4 semanas aprenderán conceptos que normalmente se enseñan en un semestre completo. Habrá momentos de frustración cuando el código no funcione, cuando los errores parezcan incomprensibles, cuando sientan que no avanzan lo suficientemente rápido.

Pero quiero que recuerden tres cosas fundamentales:

1

No están aquí para convertirse en ingenieros de software. Están aquí para entender programación lo suficiente como para inspirar a sus futuros estudiantes, para mostrarles que la tecnología no es magia inaccesible sino una herramienta que pueden dominar para cambiar su realidad.

2

Cada error es una oportunidad de aprendizaje. Los mejores programadores del mundo pasan más tiempo depurando código que escribiéndolo. La diferencia es que ellos han aprendido a ser pacientes y sistemáticos. Ustedes también lo lograrán.

3

El proyecto que desarrollarán puede tener un impacto real. El sistema de seguimiento de estudiantes en riesgo no es un ejercicio académico abstracto - es una herramienta que podrían implementar en sus escuelas para ayudar a prevenir la deserción, para identificar a tiempo cuando un estudiante necesita apoyo.

Guatemala necesita docentes de tecnología que no solo enseñen a usar herramientas, sino que formen pensadores computacionales capaces de ver problemas y imaginar soluciones.

Este curso es su primer paso en esa dirección. Sean pacientes consigo mismos, apoyen a sus compañeros, hagan preguntas sin miedo, experimenten, equivóquense, aprendan.

¡Vamos a programar para transformar!
